## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express

**Database:** MongoDb
## Run Locally

Clone the project

```bash
  git clone https://github.com/21A91A6152/LemonProcurement.git
```

Go to the project directory

```bash
  cd LemonProcurement
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run start
```
